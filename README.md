# ClaimGuard

Denial prevention and appeal operations SaaS for independent medical practices.

## Demo

- Practice dashboard: `practice@claimguard.demo` / `DemoPass!2026`
- Admin dashboard: `admin@claimguard.demo` / `AdminPass!2026`

## Run locally

```bash
npm install
npm run dev
```

## What is included

- Next.js landing page.
- Demo authentication for two roles.
- Practice dashboard with claim risk scoring and work queue.
- Admin dashboard with business metrics and funnel status.
- Browser persistence through `localStorage`.
- Business plan in Markdown, HTML, PDF, and Notion.
